ERRORES COMINES AL INSTALAR ANGULAR ---------------------

		Reinstalar la versión de Node.js  (abre una terminal con permisos de Administrador.)
			>nvm uninstall 24.0.0		(Elimina la versión dañada)
			>nvm install 24.0.0		(Vuelve a instalarla. Asegúrate de tener una conexión estable)

			> npm -v
		Si el error persiste, intenta instalar la versión LTS (Long Term Support)  que suele ser más estable
			>nvm install lts
			>nvm use lts
